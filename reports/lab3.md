## 1. Программы для разных действий

### 1.1 Используемые программы

В своей повседневной работе я использую следующие программы для разных целей:

1. Для редактирования текстовых файлов, которые попадаются на пути я использую **VS Code**. 

2. Для программирования на C++ я также использую **VS Code**. 

3. Для выполнения команд в консоли я использую:
- **WSL (Windows Subsystem for Linux)** 
- **Командная строка Windows (CMD)** 

4. Для навигации по файловой системе и манипуляций с файлами я использую:
- **Проводник Windows** 
- **Git Bash**
- **Командную строку Windows (CMD)** 

5. Для работы с Git  я использую :
- **Git Bash**
- **Встроенный Git в VS Code**

### 1.2 Альтернативы

Альтернативы для каждой из программ для каждой цели:
1. Для редактирования текстовых файлов можно использовать: **Sublime Text**, **Vim** или **Notepad++**.

2. Для программирования на C++ можно использовать **Visual Studio**.

3. Для выполнения команд в консоли можно использовать **PowerShell**.

4. Для навигации по файловой системе и манипуляций с файлами можно использовать **Total Commander**.

5. Для работы с Git  можно использовать **GitHub Desktop** или **GitKraken**.

В первом семестре я использовала **Sublime Text** для написания программ на C++. В нем было легко разобраться, и он отлично подходил для небольших учебных программ. Но по мере усложнения лабораторных работ возможностей **Sublime Text** стало не хватать. Так, в нем нет встроенного терминала и поддержки отладки. Со второго семестра я перешла на **VS Code**, в котором есть много полезных функций.

## 2. Look and Feel

### 2.1 Цвета интерфейса и шрифты

**Чтобы изменить тему оформления в  **VS Code**, необходимо:**
1. Нажать сочетание клавиш `Ctrl+K Ctrl+T` (или через меню: `File` -> `Preferences` -> `Color Theme`)
2. Выбрать нужную тему из списка

Кроме того, в **VS Code** есть возможность установить дополнительные темы, для этого необходимо:
1. Нажать сочетание клавиш `Ctrl+Shift+X` (или нажать на иконку `Extensions` в левой боковой панели)
2. В поиске ввести `theme` 
3. Выбрать понравившуюся тему
4. Нажать `Install`
5. Нажать сочетание клавиш `Ctrl+K Ctrl+T`, чтобы выбрать установленную тему из списка

**Чтобы изменить шрифт и его размер, необходимо:**
1. Открыть настройки с помощью сочетания клавиш `Ctrl+,` (или через меню: `File` -> `Preferences` -> `Settings`)
2. В поиске ввести `Font Size` — появится поле для изменения размера шрифта
3. Ввести `Font Family` — можно указать название шрифта
4. Изменения применятся сразу после сохранения

В итоге я установила тему оформления **Light Pink Theme**, которая задает общую цветовую гамму программы, а размер шрифта оставила **14px**, так как он наиболее приятный для чтения.

### 2.2 Расположение элементов интерфейса

**VS Code** позволяет настраивать расположение всех панелей. Вот основные способы:

**Основные способы настройки:**

1. **Быстрое разделение экрана:**
   - Справа сверху есть кнопка **Split Editor** — нажав на неё, можно разделить редактор на несколько окон
   - Также можно перетаскивать вкладки мышью в нужную сторону

2. **Подробная настройка через меню:**
   - `View` -> `Appearance` — здесь можно управлять видимостью и положением всех панелей:
     - Показать/скрыть Activity Bar, Side Bar, Panel, Status Bar
     - Переместить боковую панель вправо/влево
     - Переместить нижнюю панель вверх/вниз
     - Включить Secondary Side Bar (вторую боковую панель)
     - Настроить Menu Bar

3. **Горячие клавиши для быстрого доступа:**
   - `Ctrl+B` — показать/скрыть боковую панель
   - `Ctrl+J` — показать/скрыть нижнюю панель
   - `Ctrl+\` — разделить редактор по вертикали

В итоге я переместила боковую панель вправо через `View` -> `Appearance` -> `Move Primary Side Bar Right` и разделила экран на две колонки через кнопку **Split Editor**, чтобы видеть код и тесты одновременно

## 3. Эргономика работы с кодом

**Чтобы открыть список горячих клавиш в VS Code, необходимо:**
- Перейти в `File` -> `Preferences` -> `Keyboard Shortcuts`(или `Ctrl+K Ctrl+S`)

В этом окне можно:
- Просматривать все команды и их сочетания
- Искать по названию команды или по клавишам
- Менять существующие сочетания
- Добавлять свои сочетания

**Несколько полезных горячих клавиш:**
- `Ctrl+Enter` — вставить строку ниже и перейти на нее
- `Ctrl+Shift+Enter`— вставить строку выше
- `Alt+ Down Arrow` / `Alt+Up Arrow` — переместить строку вниз/вверх
- `Shift+Alt+ Down Arrow` / `Shift+Alt+Up Arrow` — скопировать строку вниз/вверх

**Горячие клавиши для быстрого перемещения по коду:**
- `Ctrl+Shift+F` — поиск по содержанию файлов (или через меню: `View` -> `Search`)
- `F12` — перейти к объявлению функции или типа (или через меню: правый клик на имени -> `Go to Definition`)
- `Ctrl+F12` — перейти к реализации функции или типа (или через меню: правый клик на имени -> `Go to Implementation`)
- `Shift+F12` — найти все места, где используется переменная, функция или тип (или через меню: правый клик на имени -> `Find All References`)
- `F2` — переименовать все вхождения переменной, функции или типа (или через меню: правый клик на имени -> `Rename Symbol`)
- `Alt+Left Arrow`/`Alt+Right Arrow` — быстро перемещаться между несколькими интересными местами кода (или через меню: `Go` -> `Back` / `Forward`)
- `Ctrl+/`(для строки) и `Shift+Alt+A`(для блока) — комментировать и раскомментировать большие части кода

## 4. Кастомизация процессов

В **VS Code** можно создавать собственные сочетания клавиш через графический интерфейс

**Чтобы добавить новую горячую клавишу, необходимо:**
1. Перейти в  `File` -> `Preferences` -> `Keyboard Shortcuts` (или `Ctrl+K Ctrl+S`)
2. В поиске найти нужную команду
3. Нажать на иконку **карандаша** слева от команды
4. Нажать желаемое сочетание клавиш
5. Нажать `Enter` для сохранения

**Настройка сочетания клавиш:**
- `Ctrl+1` теперь отвечает за сборку программы в нужной среде
- `Ctrl+2` теперь отвечает за запуск программы в режиме отладки
- `Ctrl+3` теперь отвечает за запуск программы в режиме релиза
- теперь отвечает за запуск автоматических теста проекта
